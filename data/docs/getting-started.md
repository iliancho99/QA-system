# Getting Started

Welcome to the QA System. This guide walks you through the basics of setting up
and asking your first question.

## Installation

Install the application by creating a virtual environment and installing the
listed dependencies. Once installed, copy the example environment file and adjust
any values for your setup.

## Asking a question

Type your question into the input box and submit it. The system retrieves the
most relevant passages from the document corpus and uses them to generate an
answer. Answers include references to the sections they were drawn from.

## Troubleshooting

If you do not receive an answer, confirm that the language model provider is
running and that documents have been ingested. Most issues come from an empty
document store or an unreachable provider.
